## 3. SEO 健康度监控

> 📊 **核心功能**：实时监控所有域名的 SEO 健康状况

### 3.1 总览统计

#### 平均健康度
- **评分范围**：0-100 分
- **等级划分**：
  - 🟢 优秀（80-100）
  - 🟡 良好（60-79）
  - 🟠 一般（40-59）
  - 🔴 较差（0-39）

#### 收录页面统计
- 已收录页面数 / 总页面数
- 收录率百分比

#### 爬取活动
- 最近 24 小时爬虫访问次数
- TOP10 关键词数量

### 3.2 域名健康度卡片

每个域名显示：

#### 基本信息
- 域名地址
- 站点名称
- 主域名标识
- 关联的主要标签

#### 健康度评分
**评分算法**：
```
健康度 = 收录分数(30分) + 爬取分数(30分) + 排名分数(25分) + 多样性分数(15分)

- 收录分数：收录率越高分数越高（满分30）
- 爬取分数：24h爬取次数越多分数越高（满分30）
- 排名分数：TOP10关键词越多分数越高（每个5分，满分25）
- 多样性分数：独立爬虫种类越多分数越高（每种5分，满分15）
```

#### 性能指标
- ✅ **已收录**：被搜索引擎收录的页面数
- 🕷️ **爬取次数**：最近 24 小时的访问次数
- 🔝 **TOP10 关键词**：排名前10的关键词数量

#### 问题诊断
系统自动诊断以下问题：
- ⚠️ 收录率偏低（< 50%）
- ⚠️ 最近24小时无爬虫访问
- ⚠️ 爬取活跃度较低（< 5次）
- ⚠️ 暂无 TOP10 关键词排名
- ⚠️ 爬虫多样性不足（< 2种）

#### 快捷操作
- **查看 Sitemap** - 跳转到网站地图管理
- **查看蜘蛛日志** - 查看详细爬虫访问记录
- **查看关键词** - 查看该域名的关键词排名

## 4. 网站管理

### 4.1 添加新网站

1. 点击「创建网站」按钮
2. 填写基本信息：
   - **网站名称**：例如"Telegram 中文站"
   - **域名**：例如"tg-cn.com"（不含 http://）
   - **网站描述**：简短描述网站定位
3. 配置 SEO 信息：
   - **SEO 标题**：搜索结果显示的标题
   - **SEO 描述**：搜索结果显示的描述
   - **SEO 关键词**：多个关键词用逗号分隔
4. 第三方集成（可选）：
   - Google Analytics ID
   - Google Search Console ID
   - 百度统计 ID
5. 点击「保存」

### 4.2 管理域名别名

> 🎯 **多域名功能**：一个网站可以绑定多个域名，每个域名显示相同内容但有不同的 SEO 配置

#### 为什么使用多域名？
- **蜘蛛池优化**：多域名提高搜索引擎爬取频率
- **地域定位**：不同域名针对不同地区
- **关键词分布**：不同域名关注不同关键词

#### 添加域名别名
1. 进入网站详情页
2. 点击「域名别名」标签
3. 点击「添加域名」
4. 填写信息：
   - **域名**：例如"tg-download-cn.com"
   - **站点名称**：该域名的独立名称
   - **站点描述**：该域名的独立描述
   - **主域名**：是否设为主域名
   - **Primary Tags**：该域名主要关注的标签/关键词
   - **Secondary Tags**：次要标签
5. 保存

#### 域名状态
- **ACTIVE**：激活，正常使用
- **INACTIVE**：停用，不显示
- **PENDING**：待配置 DNS

## 5. 文章管理

> 📝 **核心功能**：统一管理所有网站的内容，支持 AI 优化和多网站同步

### 5.1 文章列表概览

#### 统计卡片
- **文章总数**：已发布和草稿的总数
- **平均 SEO 分数**：所有文章的平均质量评分
- **SEO 完整度**：Meta 数据完整的文章比例
- **最近活动**：最近7天新增的文章数

#### 筛选器
- **按状态筛选**：全部/已发布/草稿/定时/归档
- **按 SEO 质量筛选**：
  - 全部质量
  - 优秀（≥80分）
  - 良好（60-79分）
  - 一般（40-59分）
  - 较差（<40分）
  - 缺少元数据

### 5.2 SEO 评分算法

**满分 100 分，评分标准：**

| 项目 | 满分 | 评分规则 |
|-----|------|---------|
| Meta 标题 | 20分 | 30-60字符满分，有标题10分 |
| Meta 描述 | 20分 | 120-160字符满分，有描述10分 |
| 关键词 | 20分 | ≥5个满分20分，≥3个15分，>0个5分 |
| 内容长度 | 20分 | ≥2000字满分，≥1000字15分，≥500字10分 |
| 标签 | 10分 | ≥3个满分，≥2个7分，>0个3分 |
| 封面图 | 10分 | 有封面图10分 |

#### SEO 质量指示器
文章列表中显示缺失的元数据：
- **T** = 缺少 Meta 标题
- **D** = 缺少 Meta 描述
- **K** = 缺少关键词（<3个）

### 5.3 创建文章

1. 点击「创建文章」
2. 填写基本信息：
   - **标题**：文章标题
   - **别名 (Slug)**：URL 友好的名称（自动生成）
   - **所属网站**：选择发布到哪个网站
   - **分类**：选择或创建分类
   - **标签**：添加相关标签（逗号分隔）
3. 编写内容：
   - 使用 Markdown 或富文本编辑器
   - 建议内容 >500 字
4. SEO 优化：
   - **Meta 标题**：建议 30-60 字符
   - **Meta 描述**：建议 120-160 字符
   - **关键词**：添加 3-5 个相关关键词
5. 选择状态：
   - **草稿**：保存但不发布
   - **已发布**：立即发布
   - **定时发布**：设置发布时间
6. 点击「保存」

### 5.4 文章同步

> 🔄 **多网站同步**：一篇文章可以同步到多个网站

1. 在文章列表中，点击「同步」
2. 选择要同步到的网站（可多选）
3. 点击「确认同步」
4. 系统会自动：
   - 复制文章内容
   - 调整 SEO 设置
   - 发布到目标网站

**同步状态**：
- 已同步到 X 个网站
- 点击查看同步详情

1. 根据域名的 Primary Tags 筛选文章

#### 提交到其他搜索引擎
- Bing: https://www.bing.com/ping?sitemap=[您的sitemap网址]

## 8. 蜘蛛池监控

> 🕷️ **核心功能**：监控搜索引擎爬虫访问，分析爬取模式

### 8.1 视图切换

#### 总览视图
- 全局统计数据
- 爬虫类型分布
- 访问趋势图

#### 按域名统计
- 每个域名独立统计
- 域名健康度评分
- 爬虫分布情况

### 8.2 总览统计

- **总访问次数**：选定时间范围内的总访问
- **独立爬虫数**：不同类型的爬虫数量
- **平均访问间隔**：爬虫访问的平均时间间隔

**时间范围筛选**：
- 最近 1 小时
- 最近 24 小时
- 最近 7 天
- 最近 30 天

### 8.3 爬虫类型

系统识别的主要爬虫：
- 🤖 **Googlebot** - Google 搜索爬虫
- 🦅 **Bingbot** - Bing 搜索爬虫
- 🔍 **Baiduspider** - 百度搜索爬虫
- 🦎 **YandexBot** - Yandex 搜索爬虫
- 🕷️ **其他爬虫** - 其他识别的爬虫

### 8.4 域名统计视图

每个域名显示：
- **总访问次数**：该域名的总访问
- **健康度评分**：基于访问量和爬虫多样性
- **最后访问**：最近一次爬虫访问时间
- **最后爬虫**：最近访问的爬虫类型
- **爬虫分布**：
  - 不同爬虫的访问次数
  - 占比进度条显示
  - 颜色区分不同爬虫

### 8.5 健康度评分算法

```
健康度 = 访问量分数(40分) + 爬虫多样性(30分) + 访问时效性(30分)

- 访问量分数：访问次数越多分数越高（50次=满分40）
- 爬虫多样性：独立爬虫种类越多分数越高（每种10分，满分30）
- 访问时效性：
  * 1小时内访问 = 30分
  * 24小时内 = 20分
  * 7天内 = 10分
  * 7天外 = 0分
```

## 10. 常见问题（FAQ）

### Q1: 如何提高网站 SEO 健康度？

**A**: 从以下几个方面入手：
1. **提高收录率**：
   - 定期发布高质量内容
   - 确保每篇文章 SEO 数据完整
   - 主动提交 Sitemap

2. **增加爬取频率**：
   - 配置多个域名别名
   - 定期更新内容
   - 提高内容质量

3. **优化关键词排名**：
   - 选择长尾关键词
   - 优化 Meta 标签
   - 建立内部链接

4. **提升爬虫多样性**：
   - 提交到多个搜索引擎
   - 使用多域名策略
   - 提高网站权重

### Q2: 为什么我的文章 SEO 分数低？

**A**: 检查以下项目：
- [ ] Meta 标题是否在 30-60 字符？
- [ ] Meta 描述是否在 120-160 字符？
- [ ] 是否添加了 3+ 个关键词？
- [ ] 文章内容是否 >500 字？
- [ ] 是否添加了 2+ 个标签？
- [ ] 是否上传了封面图？

每项满足可获得相应分数，总分 100 分。

### Q3: 如何使用 AI 优化功能？

**A**:
1. 前往「系统设置」→「API 配置」
2. 配置 OpenAI API Key
3. 在文章编辑页面，点击「AI 优化」
4. 系统会自动：
   - 优化 Meta 标题和描述
   - 生成相关关键词
   - 分析内容质量
   - 提供改进建议

### Q4: 多域名功能如何工作？

**A**:
1. **一个网站，多个域名**：
   - 所有域名共享相同的文章内容
   - 每个域名有独立的 SEO 配置
   - 每个域名有独立的 Primary Tags

2. **内容筛选**：
   - 根据域名的 Primary Tags 自动筛选文章
   - 只有包含相关标签的文章才会出现在该域名

3. **SEO 优势**：
   - 增加曝光度
   - 提高爬取频率
   - 分散关键词竞争

### Q5: 如何同步文章到多个网站？

**A**:
1. 在「文章管理」列表中，找到要同步的文章
2. 点击「同步」按钮
3. 选择目标网站（可多选）
4. 点击「确认同步」
5. 系统会自动复制并发布到目标网站

**注意**：
- 同步是单向的（源 → 目标）
- 同步后的文章独立管理
- 修改源文章不会自动更新已同步的文章

### Q6: 如何解读爬虫访问数据？

**A**:
- **高访问频率** = SEO 健康，搜索引擎关注度高
- **低访问频率** = 需要优化内容，提交 Sitemap
- **多种爬虫** = 多搜索引擎覆盖，效果好
- **单一爬虫** = 需要扩展到其他搜索引擎

**建议行动**：
- 访问次数 < 5/24h：提交 Sitemap，发布新内容
- 爬虫种类 < 2：手动提交到其他搜索引擎
- 访问间隔 > 7天：检查网站是否正常，内容是否更新

### Q7: 关键词排名不准确？

**A**:
- 关键词排名需要手动或自动追踪
- 系统记录的是历史排名数据
- 不同地区、个性化因素会影响排名
- 建议使用第三方工具交叉验证

### Q8: 如何备份数据？

**A**:
**方式一：宝塔面板自动备份**
1. 登录宝塔面板
2. 「数据库」→「备份」
3. 设置自动备份计划

**方式二：手动导出**
```bash
pg_dump -U seo_user -d seo_websites > backup.sql
```
